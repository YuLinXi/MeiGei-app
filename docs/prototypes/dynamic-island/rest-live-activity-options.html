A 竞赛计分板

综合最稳，训练数据可读性最好，建议作为默认方案。

B 教练提示型

适合降低用户理解成本，compact 用下一组数据而非倒计时。

C 左右分栏型

信息多，但小屏上更容易挤，需要真机验证圆角遮挡。

D 进度轨道型

系统感强，动效空间更好，适合作为 A 的轻动效版本。

E 极简命令型

工程实现简单，品牌感弱，更适合作为低风险备选。
